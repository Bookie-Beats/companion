# Bookie Beats Companion - Distribution Branch

This branch contains the built extension files ready for installation.

## Quick Download & Install

### For Chrome, Edge, Brave, or Opera:

1. **Click the green "Code" button** above → **"Download ZIP"**
2. **Unzip the downloaded file** on your computer
3. **Open your browser** and go to `chrome://extensions/`
4. **Turn on "Developer mode"** (toggle switch in top-right)
5. **Click "Load unpacked"** button
6. **Select the "chrome" folder** from the unzipped files
7. **Visit kalshi.com** - you should see the BB Companion badge!

### For Firefox:

1. **Click the green "Code" button** above → **"Download ZIP"**
2. **Unzip the downloaded file** on your computer
3. **Open Firefox** and go to `about:debugging#/runtime/this-firefox`
4. **Click "Load Temporary Add-on..."** button
5. **Navigate to the "firefox" folder** from the unzipped files
6. **Select the "manifest.json" file** inside the firefox folder
7. **Visit kalshi.com** - you should see the BB Companion badge!

> **Note for Firefox users**: Temporary add-ons are removed when you close Firefox. Consider using Firefox Developer Edition for longer persistence.
